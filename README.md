# PIC Digital Cooker

This project simulates a **digital cooker system** using a PIC microcontroller.

It allows the user to control cooking time and modes through buttons, while displaying information on an LCD. The system mimics the basic behavior of a real digital cooker.

---

## 🚀 Overview

The system works as follows:

* User inputs settings using buttons
* The selected mode and time are shown on an LCD
* The system counts down the cooking time
* Output behavior changes depending on the selected mode

---

## ⚙️ Features

* LCD display for user interaction
* Button-based input system
* Timer-based countdown logic
* Multiple cooking modes
* Real-time updates on screen
* Simple user interface

---

## 🛠️ Tech Stack

* PIC Microcontroller
* Embedded C
* MPLAB / XC8 
* LCD module
* Push buttons

---

## ⚙️ How it works

1. User selects a cooking mode
2. User sets the desired time
3. The system starts counting down
4. LCD displays:

   * current mode
   * remaining time
5. When time finishes, the system stops
